# 清空数据库操作指引

## 方法一：通过微信开发者工具云开发控制台（推荐）

### 步骤：

1. **打开微信开发者工具**
   - 打开你的小程序项目

2. **进入云开发控制台**
   - 点击顶部菜单栏的"云开发"按钮
   - 选择你的云环境：`cloudbase-0grjcjxkb55037a3`

3. **清空 activities 集合**
   - 在左侧菜单点击"数据库"
   - 找到 `activities` 集合
   - 点击集合名称进入详情页
   - 点击"数据"标签页
   - 点击"删除"按钮（或选中所有数据后批量删除）
   - 确认删除

4. **清空 bills 集合**
   - 找到 `bills` 集合
   - 重复上述步骤，清空所有数据

## 方法二：通过云数据库控制台网页版

1. **访问云开发控制台**
   - 打开：https://console.cloud.tencent.com/tcb
   - 登录你的腾讯云账号

2. **选择云环境**
   - 找到环境 ID：`cloudbase-0grjcjxkb55037a3`
   - 点击进入

3. **清空集合数据**
   - 进入"数据库"页面
   - 分别清空 `activities` 和 `bills` 集合的所有数据

## 方法三：使用小程序代码临时清空（仅用于开发测试）

如果需要通过代码清空，可以在小程序控制台执行以下代码：

```javascript
// 在微信开发者工具的控制台（Console）中执行
const db = wx.cloud.database();

// 清空 activities 集合
db.collection("activities").get().then(res => {
  const promises = res.data.map(item => 
    db.collection("activities").doc(item._id).remove()
  );
  return Promise.all(promises);
}).then(() => {
  console.log("activities 清空完成");
  
  // 清空 bills 集合
  return db.collection("bills").get();
}).then(res => {
  const promises = res.data.map(item => 
    db.collection("bills").doc(item._id).remove()
  );
  return Promise.all(promises);
}).then(() => {
  console.log("bills 清空完成");
  console.log("所有数据已清空");
});
```

## 注意事项

⚠️ **警告：清空数据操作不可恢复！**

- 清空前请确保已备份重要数据（如有需要）
- 建议在开发/测试环境操作，生产环境请谨慎
- 清空后，所有活动报名记录和记账记录将永久删除

## 验证清空结果

清空完成后，可以：
1. 在小程序中查看"活动管理"页面，应该显示"暂无活动"
2. 在"查询"页面查看，应该显示"暂无活动报名记录"和"暂无 AA 记账记录"
3. 在云开发控制台的数据库中，两个集合应该为空
