# dragonReserveSystem 代码总结报告

## 项目概述
- **项目名称**: 龙城俱乐部小程序
- **项目类型**: 微信小程序 + 云开发
- **代码规模**: 2.5MB, 52个核心文件
- **主要功能**: 俱乐部活动管理、签到、记账、成员管理

## 一、技术架构

### 前端架构 (微信小程序)
```
技术栈: WXML + WXSS + JavaScript
框架: 微信小程序原生框架
状态管理: 全局App数据 + 本地存储
UI组件: 微信原生组件
```

### 后端架构 (云开发)
```
云函数: 6个核心云函数
数据库: 微信云开发数据库
集合: activities, users 等
认证: 微信OpenID + 自定义权限系统
```

## 二、目录结构分析

### 核心目录
```
dragonReserveSystem/

│   ├── app.json         # 应用配置、页面路由、窗口设置
```
