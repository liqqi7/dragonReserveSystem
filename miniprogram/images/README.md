# TabBar 图标说明

请在此目录下放置以下图标文件：

1. **活动管理图标**：
   - `activity.png` - 未选中状态（建议尺寸：81x81px）
   - `activity-active.png` - 选中状态（建议尺寸：81x81px）

2. **记账图标**：
   - `accounting.png` - 未选中状态
   - `accounting-active.png` - 选中状态

3. **排行榜图标**：
   - `history.png` - 未选中状态
   - `history-active.png` - 选中状态

## 图标建议

- **活动管理**：可以使用日历、列表、活动相关的图标
- **记账**：可以使用计算器、钱包、记账本相关的图标
- **排行榜**：可以使用搜索、统计、图表相关的图标

## 临时方案

如果暂时没有图标文件，可以先使用纯文字显示（已在 app.json 中配置）。
