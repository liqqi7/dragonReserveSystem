# TabBar 图标说明

请在此目录下放置以下图标文件：

1. **主页图标**：
   - `tab-home.jpg` - 未选中状态
   - `tab-home-active.jpg` - 选中状态

2. **记账图标**：
   - `accounting.png` - 未选中状态
   - `accounting-active.png` - 选中状态

3. **排行榜图标**：
   - `tab-ranking.jpg` - 未选中状态
   - `tab-ranking-active.jpg` - 选中状态

4. **我的图标**：
   - `tab-profile.jpg` - 未选中状态
   - `tab-profile-active.jpg` - 选中状态

## 图标建议

- **主页**：可以使用首页、导航、活动相关的图标
- **记账**：可以使用计算器、钱包、记账本相关的图标
- **排行榜**：可以使用搜索、统计、图表相关的图标
- **我的**：可以使用用户、个人中心相关的图标

## 临时方案

如果暂时没有图标文件，可以先使用纯文字显示（已在 app.json 中配置）。
